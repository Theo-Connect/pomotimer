# Pomotimer

Flutter로 제작한 포모도로 타이머 앱입니다.

## 기능

- 25분 포모도로 타이머
- 시작/일시정지 기능
- 완료된 포모도로 세션 카운터
- 직관적인 UI/UX

## 기술 스택

- **Flutter** - 크로스 플랫폼 앱 개발
- **Dart** - 프로그래밍 언어

## 프로젝트 구조

```
lib/
├── main.dart          # 앱 진입점 및 테마 설정
└── screens/
    └── home_screen.dart   # 메인 타이머 화면
```

## 실행 방법

1. Flutter SDK 설치
2. 프로젝트 클론
   ```bash
   git clone https://github.com/giga2roo/pomotimer.git
   cd pomotimer
   ```
3. 의존성 설치
   ```bash
   flutter pub get
   ```
4. 앱 실행
   ```bash
   flutter run
   ```
